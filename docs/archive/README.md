# docs/archive

本目录保存历史材料，避免它们继续混在活跃文档入口里造成漂移。

归档文件不是当前项目基线。若这里的内容与以下文件冲突，一律以后者为准：

1. `AGENTS.md`
2. `docs/PROJECT_BASELINE.md`
3. `docs/STANDARD_PATCH_FRAMEWORK.md`
4. `docs/CURRENT_STATE.md`
5. 活跃策略语义基线文档

目录说明：

- `legacy/`：旧协作入口、旧规则、旧引用文件。
- `reports/`：阶段性审计报告。
- `scratch/`：临时笔记、系统残留、一次性现场材料。

不要从 `scratch/` 中复制命令、token 或生产操作指令直接执行。
